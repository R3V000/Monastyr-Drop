# Prezentacja Wydarzenia Wakacyjnego

Statyczna prezentacja wydarzenia wakacyjnego Monastyr2 przygotowana w Next.js.

## Komendy

```bash
npm install
npm run dev
npm run export
npm run preview
```

Po `npm run export` gotowe pliki statyczne znajda sie w katalogu `out/`.
`npm run preview` serwuje katalog `out/` lokalnie pod `http://localhost:3001`.
